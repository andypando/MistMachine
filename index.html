Welcome to Mist Machine

Welcome to the Mist Machine platform. This service allows for the rapid develop of
"sideware" that will interact with the Mist API to perform operations that are not
currently supported in the portal.

Use the navigation menu on the left to access various system functions.

Build sites from CSV

Allows the user to upload a .csv file with a name, address and note field. Order of the fields is
not important, in the second step you select the column to use for each element.
Working on API call to validate the address and return geo-location so maps are correct in portal.
Note: Working on a small bug, only works for Global 01 currently

Set Autoupgrade per Site

Allows the user to provide Org. ID and API Token to querry Org for list of sites. Then you have
the option to select the site to assign auto-upgrade firmware versions to. For each AP model you set
version you want, set the upgrade time, and set the upgrade day.

Get Complete Inventory Reports

Allows the user to provide Org. ID and API Token to querry Org for list of all inventory. It will then
sort the output into seperate .csv files based on type. This is helpful when you have switches in a VC
as the output from the Mist portal is not usable in the format provided.

Custom Site Upgrades

Allows the user to provide Org. ID and API Token to querry Org for list of sites. You can then select
the sites and upgrade method you wish to use on those sites. Big Bang, Serial, Canary or RRM
Note: Only Big Bang has been tested and it works, still need to test the other methods

Bulk Site Delete

Allows the user to provide Org. ID and API Token to querry Org for list of sites. You can then select
the ones you would like to delete from the Org
